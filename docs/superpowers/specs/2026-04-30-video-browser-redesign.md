# 视频浏览器改造设计

## 目标

将原有的简单视频播放 Modal 改造为内置文件浏览器的视频浏览器，支持直接从"此电脑"浏览硬盘目录，仅展示视频格式文件，并美化整体 UI。

## 变更摘要

- **删除**：`VideoPlayerModal.tsx`（旧组件）
- **新增**：`VideoBrowserModal.tsx`（新组件）、`fs.ipc.ts`（文件系统 IPC）
- **修改**：`ipc-channels.ts`（新增 IPC 通道）、`ui.slice.ts`/`AppSider.tsx`（适配新组件名）

---

## 架构

| 层 | 职责 | 位置 |
|---|---|---|
| IPC 处理器 | `fs:list-drives` 获取盘符列表，`fs:read-dir` 读取目录内容（仅返回视频文件 + 子目录） | `src/main/ipc/fs.ipc.ts` |
| 视频浏览器组件 | 左侧文件夹树 + 右侧视频网格 + 内嵌播放器 | `src/renderer/src/features/video/VideoBrowserModal.tsx` |
| 状态管理 | Modal 开关（已有 `videoPlayerModalOpen`）；浏览状态（当前路径、盘符列表、展开节点、选中视频）为组件内 `useState` | `ui.slice.ts`（modal 开关）+ 组件内 useState |

文件过滤（仅保留视频扩展名）在**主进程**完成，避免向前端传输非视频文件数据。

## 组件树

```
VideoBrowserModal (自定义标题栏，可拖动)
├── DriveSelector           — 按钮组切换盘符
├── FolderTree              — 左侧面板，Tree 组件
│   └── FolderNode          — 递归节点，懒加载子目录
└── VideoGrid               — 右侧面板，视频卡片网格 / 视频播放器
    ├── VideoCard           — 缩略图图标 + 文件名 + 大小
    └── <video> 播放器       — 点击卡片时替换网格显示
```

---

## 数据流

1. 弹窗打开 → 调用 `fs:list-drives` 获取盘符列表
2. 选中盘符 → 调用 `fs:read-dir` 获取根目录内容
3. 点击文件夹 → 调用 `fs:read-dir` 获取子目录内容（懒加载）
4. 点击视频文件 → 通过 `file://` 协议加载本地文件到 `<video>` 播放

---

## IPC 通道

```
FS_LIST_DRIVES: 'fs:list-drives'   → 返回 string[] (盘符列表，如 ["C:", "D:", "E:"])
FS_READ_DIR:    'fs:read-dir'      → 入参: { path: string }
                                   → 返回: { dirs: string[], videos: { name: string, size: number, path: string }[] }
```

## 交互行为

| 操作 | 行为 |
|---|---|
| 点击盘符按钮 | 右侧加载该盘符根目录的视频文件，左侧展示文件夹树 |
| 展开文件夹树节点 | 懒加载子目录，显示 loading 状态 |
| 选中文件夹 | 右侧刷新为该文件夹下的视频列表 |
| 点击视频卡片 | 卡片区收起，展开视频播放器自动播放 |
| 点击另一视频 | 切换播放 |
| 双击视频卡片 | 全屏播放 |
| 返回/关闭播放 | 播放器收起，回到卡片网格 |
| 拖拽标题栏 | 移动 Modal 位置 |
| 空文件夹 | 显示空状态："此文件夹下没有视频文件" |
| 盘符不可访问 | 显示错误提示（如无光盘的光驱） |
| 关闭弹窗 | 停止播放，重置状态 |

---

## 视频格式

前端过滤和后端均支持：`mp4, webm, ogg, mkv, avi, mov, wmv, flv`

---

## 与旧组件对比

| | 旧 VideoPlayerModal | 新 VideoBrowserModal |
|---|---|---|
| 文件来源 | 手动选择单个文件 | 浏览整个硬盘目录 |
| 过滤 | 浏览器原生 `accept="video/*"` | 后端按扩展名过滤 |
| 布局 | 单一输入栏 + 视频区 | 左侧文件夹树 + 右侧网格 + 底部状态栏 |
| 可拖动 | 不支持 | 支持（自定义标题栏拖拽） |
